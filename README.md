### **研究的过程**

1. 了解既有的答案与证据。
    * 搜寻、评估、筛选、分析、批判、整合的能力
2. 分析既有答案，评估其可信度与待理清的疑点。
    * 批判性思考的能力
3. 搜集进一步的证据，提出更可靠（改良）的答案或解决方案。
    * 创新的能力
